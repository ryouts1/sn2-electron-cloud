Reactive-only の定義

reactive-only 表示は、O / C / Cl の x 軸方向 σ チャネルに重みを置いた AO projector と、 その projector 上で最も反応性の高い occupied donor / virtual acceptor 軌道の抽出で定義しています。 つまり「反応に無関係な spectator density を UI から外す」ための表示であり、厳密な NBO 分析そのものではありません。
